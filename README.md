## 📥 نصب و اجرا

برای نصب و اجرا مراحل زیر را انجام دهید:

```bash
# نصب ابزار لازم
sudo apt install p7zip-full

# استخراج فایل
7z e SaPhotoshop.7z

# ورود به پوشه
cd SaPhotoshop/

# دادن دسترسی اجرا
chmod +x install.sh

# اجرای نصب
./install.sh
